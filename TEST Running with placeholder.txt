Run it via Docker (recommended)

Start: docker-compose up -d --build
Check health: curl http://localhost:8000/api/test
Open UI: http://localhost:8000
View logs: docker-compose logs -f email-platform and docker-compose logs -f db
Stop: docker-compose down
